void Visualizer::VisualizePoint ( sf::Vector2f pos,
sf::Color color = VIZ_DEFAULT_COLOR )
static

Visualize a point in space for debugging.

This function allows you visualize a world-space position by drawing a dot on screen.

Note
This function may be called anywhere, by anyone. This function does not need to be called inside a draw function, or by an object deriving from DrawObject.
Parameters
[in]posThe world-space position at which a dot should be drawn.
[in]colorThe color of the dot to be drawn (if this argument is omitted, Cyan will be used)
Returns
Does not return anything.

Example 1

// Player.cpp
#include "Player.h"
#include "../Engine Code/Visualizer.h"
void Player::Update(float deltaTime)
{
Visualizer::VisualizePoint(this->pos);
}
