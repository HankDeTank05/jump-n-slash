void InputObject::KeyPressed ( sf::Keyboard::Key key)
protectedvirtual

Do something when a key is pressed.

The function called by the engine whenever a key registered for key press events is pressed. It is only called on the first frame the key is held down. This function does nothing by default, and can be overridden by an InputObject -derived class. Any behavior that should occur when a key is pressed should go in the overridden function body.

Parameters
[in]keyThe key which was pressed.
Returns
Does not return anything.
