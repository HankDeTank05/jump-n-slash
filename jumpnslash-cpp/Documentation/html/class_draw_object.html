void DrawObject::Draw ( )
protectedvirtual

Draw an object.

The function called by the engine to draw an object. This function does nothing by default, and can be overridden by a DrawObject -derived class. Any behavior that should occur when drawing an object should go in the overridden function body.

Returns
Does not return anything.

void DrawObject::Render ( sf::Drawable & drawable)
protected

Render a drawable SFML object.

This function renders the sf::Drawable object passed to it. The following types are sf::Drawable -derived, and therefore may be passed to this function:

  • sf::Shape
  • sf::Sprite
  • sf::Text
  • sf::VertexArray
  • sf::VertexBuffer
Note
This function is only accessible to classes deriving from DrawObject .
Parameters
[in]drawableThe object to draw.
Returns
Does not return anything.

}
#endif

void DrawObject::RequestDrawDeregistration ( )
protected

Request that the engine stop drawing this object.

Call this function in a DrawObject -derived class to let the engien know you would like to stop drawing the object.

Note
This function is only accessible to classes deriving from DrawObject .
As the name of this function implies, this puts in a request with the engine for your object to stop being drawn. The request is processed by the engien on the same frame this fucntion is called, and calls to DrawObject::Draw will cease the following frame.
Returns
Does not return anything.
