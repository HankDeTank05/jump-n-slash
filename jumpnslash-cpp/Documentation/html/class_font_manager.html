void FontManager::LoadFont ( std::string key,
std::string filename )
static

Load a font into the engine.

Load a font into the engine. You must specify a key with which to identify the font, and the name of the file to load.

Parameters
[in]keyThe key used to identify the font. (Duplicate keys are not allowed!)
[in]filenameThe name of the file to load. The file must be in assets/fonts/ or a subfolder within it.
Note
The filename argument can include subfolders within the fonts folder. See examples below.
Returns
Does not return anything.

Example 1

The following example demonstrates how to load a font located in the fonts folder.

// LoadResources.cpp
#include "../Engine Code/JumpSlashEngine.h"
#include "../Engine Code/FontManager.h"
void JumpSlashEngine::LoadResources()
{
// load fonts
FontManager::LoadFont("arial", "arial.ttf");
FontManager::LoadFont("consolas", "consola.tff");
FontManager::LoadFont("delfino", "Delfino.ttf");
}
