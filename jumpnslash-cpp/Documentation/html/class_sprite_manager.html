sf::Texture * pTex,
sf::IntRect spriteRect )
static

Load a sprite into the engine with a texture pointer, using a part of the texture.

Load a sprite into the engine. You must specify a key with which to identify the sprite, provide a pointer to an existing texture, and provide an sf::IntRect specifying the area of the texture to be used in the sprite.

Parameters
[in]keyThe key used to identify the sprite. (Duplicate keys are not allowed!)
[in]pTexA pointer to the texture with which to create the sprite. Only the portion of the texture specified by the spriteRect will be used.
[in]spriteRectThe rectangle defining the area of the texture to be used in the sprite. This area can be smaller than the texture, or larger than it.
Returns
Does not return anything.
