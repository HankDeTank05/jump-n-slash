Load a texture into the engine. 

Load a texture into the engine. You must specify a key with which to identify the texture, and the name of the file to load.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em key} & The key used to identify the texture. (Duplicate keys are not allowed!) \\
\hline
\mbox{\texttt{ in}}  & {\em filename} & The name of the file to load. The file must be in {\ttfamily Assets/\+Textures/} or a subfolder within it. \\
\hline
\mbox{\texttt{ in}}  & {\em smooth} & Whether or not a texture should be smoothed. If omitted, this argument is {\ttfamily true} by default.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
The {\ttfamily filename} argument can include subfolders within the Textures folder. See examples below.
\end{DoxyNote}
\begin{DoxyReturn}{Returns}
Does not return anything.
\end{DoxyReturn}
