Visualize a point in space for debugging. 

This function allows you visualize a world-\/space position by drawing a dot on screen.

\begin{DoxyNote}{Note}
This function may be called anywhere, by anyone. This function does not need to be called inside a draw function, or by an object deriving from \doxylink{class_draw_object}{Draw\+Object}.
\end{DoxyNote}
