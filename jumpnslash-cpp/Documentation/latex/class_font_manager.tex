Load a font into the engine. 

Load a font into the engine. You must specify a key with which to identify the font, and the name of the file to load.


\begin{DoxyParams}[1]{Parameters}

\hline
\mbox{\texttt{ in}}  & {\em filename} & The name of the file to load. The file must be in {\ttfamily assets/fonts/} or a subfolder within it.\\
\hline
\end{DoxyParams}
