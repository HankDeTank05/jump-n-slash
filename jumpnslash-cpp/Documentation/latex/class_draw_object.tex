Request that the engine stop drawing this object. 

Call this function in a {\ttfamily \doxylink{class_draw_object}{Draw\+Object}} -\/derived class to let the engien know you would like to stop drawing the object.

\begin{DoxyNote}{Note}
This function is only accessible to classes deriving from {\ttfamily \doxylink{class_draw_object}{Draw\+Object}} .

As the name of this function implies, this puts in a request with the engine for your object to stop being drawn. The request is processed by the engien on the same frame this fucntion is called, and calls to {\ttfamily \doxylink{class_draw_object_a58db365c96dc8ab875f2a03b23e3c2ae}{Draw\+Object\+::\+Draw}} will cease {\bfseries{the}} {\bfseries{following}} frame.
\end{DoxyNote}

Does not return anything.
