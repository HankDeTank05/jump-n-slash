Update an object. 

The function called by the engine to update an object. This function does nothing by default, and can be overridden by an {\ttfamily \doxylink{class_update_object}{Update\+Object}} -\/derived class. Any behavior that should occur when updating an object should go in the overridden function body.


\begin{DoxyParams}[1]{Parameters}
\mbox{\texttt{ in}}  & {\em delta\+Time} & The amount of time since the previous frame, in seconds.\\
\hline
\end{DoxyParams}
\begin{DoxyNote}{Note}
This engine is not framerate-\/controlled! A frame will render as soon as it can, meaning the time between frames is unknown. Any behavior (such as movement) which occurs over time needs to be multiplied by the provided {\ttfamily delta\+Time} argument in order to appear normal!
\end{DoxyNote}
\begin{DoxyReturn}{Returns}
Does not return anything.
\end{DoxyReturn}
